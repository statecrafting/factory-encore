# Page type: help

A help or documentation page: guidance, FAQs, and support contact.

## Build it with

- PrimeVue `Accordion` for FAQs and `Card`/`Panel` for sections.
- Support contact details come from configuration, not hard-coded values.
- No data source required.

## Auth

Typically `view_type: public`, `requires_auth: false`. Match the audience if the
help content is specific to a signed-in area.
